# HubDiseaseAccruals
## Data-driven identification of hub chains in disease accruals
This novel nested k-means trajectory clustering method groups the disease accrual process and identifies hub chains. It was used in a recent paper: Han, S., Zhou, C., Li, S. et al. Data-driven identification of hub chains in disease accruals to reduce healthcare utilization and costs.
